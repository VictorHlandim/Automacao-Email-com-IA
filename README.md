# Automacao-Email-com-IA
Uma aplicação em C# que automatiza o envio de emails através de um agente de IA pessoal. O sistema utiliza o Ollama para processar comandos em linguagem natural do usuário e executar ações de email baseadas nas respostas da IA.

Características:
- Integração com Ollama para processamento de linguagem natural
- Automatização de envio de emails via SMTP
- Reconhecimento de nomes em endereços de email
- Processamento de comandos do usuário em português
- Resposta estruturada em JSON para diferentes ferramentas
- Pode mandar multiplos emails em um processo singular

Como usar:
1. Configure as credenciais SMTP
2. Configure a porta e endpoint do Ollama local
3. Execute a aplicação e digite sua ação desejada
4. A IA processará o comando e executará a ação correspondente
